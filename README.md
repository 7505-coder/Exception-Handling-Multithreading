# Exception Handling & Multithreading

## Features:
- Custom Exception (`StudentNotFoundException`)
- Multithreading (`Loader` with Runnable)
- Wrapper classes (Integer, Double)
- Exception handling using try-catch-finally
- StudentManager with validation + safe operations

## How to Run:
Use an online compiler like:
https://www.onlinegdb.com/online_java_compiler

Create folder structure:
model/
service/
exceptions/
threads/
app/

Copy corresponding files into correct packages.

Run `Main.java`.
